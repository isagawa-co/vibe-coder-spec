# Discovery — Lessons

## Seeded Anti-Patterns

1. **Asking too many questions at once** — overwhelms the vibe coder. One question at a time. Two max if closely related.
2. **Using a fixed script** — makes the conversation feel like a form. Follow-ups must be dynamic based on what the user already said.
3. **Not acknowledging what the user said** — always reflect back before asking the next question. Builds trust.
4. **Over-asking simple apps** — a landing page needs 2-3 questions. A real-time collaboration tool needs 6-8. Read the complexity signal.
5. **Guessing when unsure** — if the user's description is vague, ask. Never fill gaps with assumptions. Summarize what you understand and ask what's missing.
